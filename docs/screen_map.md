# Screen Map

| Screen | Route | Purpose |
|--------|-------|---------|
| Global Pulse | `/` | Daily macro dashboard — indices, commodities, FII/DII, sectors, market phase |
| Signal Scanner | `/scanner` | Weekly scan results — stocks ranked by 8-point score with expandable signals |
| Stock Deep Dive | `/stock/:symbol` | Full stock analysis — technicals, fundamentals, setups, AI summary, actions |
| Portfolio Monitor | `/portfolio` | Holdings grouped by health (healthy/warning/alert) with exit signal tracking |
| Weekly Briefing | `/briefing` | Weekend summary — market, sectors, top signals, portfolio alerts in one read |
