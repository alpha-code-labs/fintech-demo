FUNDAMENTAL DATA (Revenue, Profit, Margins, EPS, Debt, ROCE)
--------------------------------------------------------------
Source 1: yfinance (free)
Code:
  ticker = yf.Ticker("RELIANCE.NS")
  income = ticker.quarterly_income_stmt    # Revenue, Net Profit, EPS
  balance = ticker.balance_sheet           # Debt, Assets
  info = ticker.info                       # P/E, ROE, Debt/Equity

Available: Revenue, Net Profit, EPS, Operating Margin, P/E, ROE, Debt/Equity
NOT available via yfinance: ROCE (must calculate), Industry P/E
Limitation: Coverage inconsistent for smaller Indian companies

Source 2: Screener.in scraping (most complete for Indian stocks)
Method: requests + BeautifulSoup
  URL: https://www.screener.in/company/RELIANCE/consolidated/
  Data: Quarterly results, P&L, Balance Sheet, Cash Flow, ROCE,
        Promoter Holding, Industry P/E — all on one page
  Rate limit: Add 3 second delay between requests
  Note: No official API. Scraping may violate ToS.

Source 3: Financial Modeling Prep API (free tier: 250 calls/day)
  Standardized JSON, cleaner than scraping
  Registration: https://site.financialmodelingprep.com/

Recommended: yfinance for 80% of stocks, Screener.in scraping to fill gaps
Status: CONFIRMED ✅
