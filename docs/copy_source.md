- World

## Tone of Voice

- Direct, no fluff
- Financial professional register — not casual, not corporate
- Numbers are prominent — copy supports, doesn't distract
- "Your call" / "Your judgment" language used to distinguish human decisions from automation
- System describes what it found — never recommends
